Why Storj?

What benefits does decentralized storage offer over traditional services such as Dropbox or Google Drive?

What is Storj?

Storj is an open source project actively developing a completely decentralized, secure, and efficient cloud storage service that integrates a peer-to-peer protocol based on Bitcoin.

The Problem of
Centralized Data Storage

It's no secret that the last few years have been chaotic regarding the security and privacy of web users. There has been a record setting amount of data security breaches from private enterprise databases. The amount of personal data stored by private enterprises and national governments is at never before seen levels, and the security protecting it is often critically flawed.

Enormous file hosting sites like Megaupload have been shut down, depriving customers access to the cloud storage they paid for. Startling revelations have been published showing that government spy agencies can monitor anyone’s internet activity in real time and can access files or data put on seemingly secure cloud storage hosts like Dropbox and OneDrive. This is usually done for undisclosed reasons, in secret, and without any due process of law which is a clear violation of worldwide internet users' privacy.

People are afraid of using the internet, and rightly so. People are afraid of companies that offer them cloud storage, knowing that information is power, and that power is being abused. The problem boils down to one of centralization of data storage, the transmission of data in plain text, and centralized data administration firms who are required by law to breach their own customers’ privacy.

The Solution of
Decentralized Data Storage

Storj is a Decentralized Autonomous File Storage Network. The network runs on a protocol, cryptocurrency, and a suite of decentralized applications which allow users to store data in a secure and decentralized manner. It uses Bitcoin’s features of a public transaction ledger, public/private key encryption, and cryptographic hash functions for security.

Storj nodes, or average computers running the software, sell resources used to store and transfer information and are rewarded with Storjcoins or other cryptocurrencies. This allows users to run the software and be incentivised for sharing their unused hard drive space and bandwidth. Since many millions of computing device users have unused computing resources, that are going to waste, a way to convert those resources into a digital cash reward should quickly become popular.

By using the Bitcoin model, we can create a data storage system that can’t be censored, monitored, or suffer from downtime. Furthermore, it will be 10 to 100 times cheaper, faster, and more secure than traditional cloud storage services.
